Se buscan los n nomb res mas repetidos y la cantidad de veces que aparece cada uno, sabiendo que no hay dos nombres que se repitan la misma cantidad de veces.

Datos de entrada
-Primera linea: la cantidad c de chicos (2<=c<=400.000) y la cantidad n (1<=n<=600) de nombres mas frecuentes. Ambos numeros estan separados por un blanco y cada nombre ocupe 20 caracteres.
-c lineas con un nombre.

Datos de salida
-n lineas, cada una conteniendo el nombre y la cantidad de apariciones separadas por un blanco.
